# ShowTime OTT Viewership Prediction

## Overview
This project analyzes OTT platform content data to identify the factors that drive first-day viewership. Using exploratory data analysis and linear regression, the project highlights how variables such as trailer views, ad impressions, visitors, release timing, and genre influence content performance.

## Objective
To determine the key driver variables of first-day content viewership and generate actionable insights for marketing and release strategy.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
